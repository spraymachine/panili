# 🧶 PaNii Li — Progress Tracking Dashboard

A lightweight, collaborative Kanban-style task tracker for the PaNii Li crochet company. Built with love and yarn.

## Features

- **Simple login** — Select Mani, Pari, or Liki (no password)
- **Kanban board** — To Do · In Progress · Completed
- **Drag & drop** — Move cards between columns
- **Arrow buttons** — Keyboard-friendly card movement
- **Card management** — Add, edit, delete cards
- **User colors** — Each user's cards are tinted in their color (Blue, Green, Yellow)
- **Persistence** — Board state saved to localStorage

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview  # Preview production build
```

## Stack

- React 18 + TypeScript + Vite
- Tailwind CSS
- @dnd-kit (drag and drop)
- Zustand (state)
- Framer Motion (animations)
- Lucide React (icons)
- date-fns (dates)
